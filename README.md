## Project Requirements

## Installation
1. Clone this project on your home (or any other of your choice) folder.
```
$ cd ~
$ git clone https://github.com/maxfontana90/pelados-training.git
$ cd pelados-training
```

2. Install dependencies.
```
$ npm install
```

3. Start project.
```
$ npm run serve
```
